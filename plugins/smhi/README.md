# SMHI

Plugin name: smhi

## Description

Plugin for fetching weather forecasts from Sveriges Meteorologiska och Hydrologiska Institut.

## Config

    "smhi": {
        fetchInterval: <number> // delay between fetching new weather information (in milliseconds)
    }


### Config example

      "smhi": {
        fetchInterval: 300000
      }
